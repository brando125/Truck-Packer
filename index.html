Load from Google Sheet URL

In Google Sheets: File → Share → Publish to web, choose Comma-separated values (.csv), publish, and paste the URL below. The app stores this URL so Check for Updates can re-fetch it later.

If fetch fails (CORS / non-published URL), use Upload CSV instead.
